Image Converter

Convert images quickly in your browser.

Live Preview: https://image-converter-wine-seven.vercel.app

Features

• Supports JPEG, PNG, WebP
• Drag and drop
• No server upload
• Fast download

Usage

• Upload image
• Select format
• Convert and save

Install and Run

git clone <repo-url>
cd <folder>
npm install
npm run dev

Done. That’s as brief as it should be without looking lazy.
